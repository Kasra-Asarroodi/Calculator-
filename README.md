
# Simple calculator

## Description
This is a simple command-line calculator built using Python.  
It allows users to perform basic calculations by entering numbers and selecting their desired operator.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation for numbers
- Error handling for invalid input
- Prevents division by zero

## Technologies Used
- Python
- VS Code (development environment)

## How to Run
1. Make sure Python is installed on your system
2. Open the project folder in VS Code or terminal
3. Run the program using:
   python3 calculator.py

## Example of interacting with the calculator
Enter first number: 3  
Enter operator (+, -, *, /): -  
Enter second number: 9  
Result: -6
